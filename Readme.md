# HTML Essentials

## Creating an HTML document

```html
<!DOCTYPE html>
<html>
    <head>
        <title>This is a great title</title>
    </head>
    <body>
        <h3>This is a Heading</h3>
        <p>
            Lorem, ipsum dolor sit amet consectetur adipisicing elit. Vitae a cupiditate provident eaque laudantium architecto ea, illum officia ipsa, aut soluta incidunt! Repudiandae aspernatur nobis illo incidunt voluptates veniam quaerat.
        </p>
    </body>
</html>
```

## Adding Images

```html
<img src="images/image.png" alt="This is an image of a Yacht sailing in a wide sea" style="height: 500px; float: right" />
```

## Adding Hyperlinks

```html
<a href="otherpage.html">Click here!</a>
<a href="http://www.bbc.com" target="_blank">Click here!</a>
<a href="https://en.wikipedia.org/wiki/Yacht">
    <img src="images/image.png" alt="This is an image of a Yacht sailing in a wide sea" style="height: 500px; float: right" />
</a>
<a href="mailto:dave@elegantyachting.com">Mail us!</a>
```
